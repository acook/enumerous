# Enumerous

Handy functionality for your Arrays and Enumerables!

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     enumerous:
       github: acook/enumerous
   ```

2. Run `shards install`

## Usage

```crystal
require "enumerous/all"
```
